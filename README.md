# github-portfolio

Osobní portfolio web na GitHub Pages: https://kubica91.github.io/github-portfolio/

Domovská stránka, CV, Unity WebGL hry a 3D šachy v Three.js.

## Tech stack

React 18 + TypeScript, Vite 5, Tailwind, Three.js, Unity WebGL, i18next, Vitest.

## Spuštění

```bash
pnpm install
pnpm dev
```

Dev server běží na `http://localhost:5173/github-portfolio`.

## Deploy

Automatický přes GitHub Actions ([.github/workflows/deploy.yml](.github/workflows/deploy.yml)) při push do `main`.
